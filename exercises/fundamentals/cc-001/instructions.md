# Hello Claude Code

Welcome to your first exercise! This is a warm-up to make sure everything is
working and to get you comfortable talking to Claude.

## Your Task

Ask Claude to create a file called `hello.md` in your workspace directory.

The file should contain:
- A greeting message
- At least one interesting fact about Claude Code

That's it — just talk to Claude in plain English. For example, you could say:

> "Create a file called hello.md with a greeting and a fun fact about Claude Code."

## Requirements

Your `hello.md` file must:
1. Exist in this workspace directory
2. Have at least 3 lines of content
3. Mention "Claude" somewhere in the text

## When You're Done

Run `/cclab:check` to validate your work.
