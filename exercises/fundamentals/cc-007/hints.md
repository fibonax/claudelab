# Hints for cc-007: Command Center

## Hint 1

Claude Code has many built-in commands. Start by typing `/help` to see what's
available. Your goal is to discover and document what you find in a file called
`commands.md`.

## Hint 2

Run `/help` in Claude Code to see the list of commands. Pick at least 5
commands (including `/help` and either `/compact` or `/clear`) and write a
short description of each in `commands.md`. Each command should be on its own
line starting with `/`.

## Hint 3

Create `commands.md` with entries like this:

```
# Claude Code Commands

/help - Shows available commands and usage information
/compact - Compresses conversation context to free up space
/clear - Resets the conversation and starts fresh
/init - Initializes a CLAUDE.md file for the project
/memory - Manages Claude's persistent memory
```

List at least 5 commands with descriptions. Make sure each command line starts
with `/` and the file has at least 8 lines total.
